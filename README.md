# Password Security Toolkit

A CLI tool to analyze password strength and check if passwords have appeared in known data breaches.

## Features (planned)
- 🔍 Password strength analysis (length, charset, entropy)
- ⏱️  Crack-time estimation
- 🚨 Breach detection via HaveIBeenPwned API (using k-anonymity)
- 🔒 Password input is hidden (never echoed or logged)
